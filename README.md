# UBMarket - University of Botswana Student Marketplace

A full-stack student marketplace platform for buying, selling, and trading textbooks, rooms, and tutoring services within the UB community.

## Features

- 🔐 **Secure Authentication** - Only @ub.bw emails can register
- 📚 **Textbook Trading** - Buy, sell, and swap used textbooks
- 🏠 **Room Rentals** - Find off-campus accommodation
- 🧑‍🏫 **Peer Tutoring** - Find tutors or offer tutoring services
- 💬 **Real-time Messaging** - Chat with buyers and sellers
- ⭐ **Ratings & Reviews** - Build trust with student reviews
- 📱 **Mobile Responsive** - Works on all devices
- 💰 **Monetization** - Featured listings, premium tutors, verified badges

## Tech Stack

### Backend
- Node.js with Express
- Neon PostgreSQL
- JWT Authentication
- Bcrypt for password hashing
- Nodemailer for emails
- Multer for file uploads

### Frontend
- Vanilla HTML, CSS, JavaScript
- Responsive design with CSS Grid/Flexbox
- Fetch API for backend communication
